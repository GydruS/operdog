Структура файлов и папок проекта:

/                   <- в корне хранятся только индексный файл, автозагрузчик, конфиг сайта и константы, которые не относятся к настройкам системы (нужны для облегчения восприятия кода)
/core               <- собственно, место, где хранятся файлы системы
/core/classes       <- базовые и абстрактные, а также дополнительные классы, которые используются в системе, но которые не зависят от самой системы
/core/db_adapters   <- адаптеры для баз данных
/core/interfaces    <- интерфейсы, которые используются в системе
/core/view_adapters <- адаптеры для вьюшек
/lib                <- место для вспомогательных файлов и сторонних либ
/modules            <- папка, где хранаятся модули сайта(веб-приложения)
